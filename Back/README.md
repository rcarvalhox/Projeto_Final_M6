#  Projeto Módulo 6

## Nosso último projeto foi abordado sobre site de games digitais, onde tem a ideia de oferece jogos com preços baixos e jogos gratuitos toda semana.

### Para executar site é necessário incluir esses comandos

iniciar o backend:

- cd back
- npm install
- cd src
-node server.js

iniciar o frontend:

-cd front
-npm install
-npm start
